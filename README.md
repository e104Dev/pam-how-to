# Programação de Aplicativos Mobile - I

## How To
Este projeto tem como objetivo fornecer suporte para os alunos em situações específicas no decorrer do componente curricular `Programação de Aplicativos Mobile`.

O documento será atualizado conforme a necessidade de apoia às atividades realizadas em sala de aula, em apoio às aulas demonstrativas que trambém estarão disponibilizadas na plataforma `Planner`.

**Não deixem de sugerir inclusões e melhorias neste documento.**

### Configuração de Ambiente de desenvolvimento
O ambiente de desenvolvimento de aplicativos Android possui alguns requisitos que precisamos considerar, os requisitos estão na sessão `System requirements` [nesta página](https://developer.android.com/studio?hl=pt&gclid=Cj0KCQjw0emHBhC1ARIsAL1QGNfn8OaPBpOI-vXUNA4OsAUfhO6m_CY08Ft1FzqB6hZ-XdBtZhMyZ-waArwdEALw_wcB&gclsrc=aw.ds).
